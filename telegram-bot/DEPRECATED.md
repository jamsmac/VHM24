# ⚠️ DEPRECATED - Standalone Telegram Bot

## Статус: УСТАРЕЛ

Этот standalone Telegram-бот **больше не используется** и был удален из `docker-compose.yml`.

## Причина удаления

Проект содержал **два конфликтующих Telegram-бота**:

1. **Этот (telegram-bot/)** - простой standalone сервис
2. **Интегрированный (backend/src/modules/telegram/)** - полноценный модуль с базой данных

При использовании одного токена (`TELEGRAM_BOT_TOKEN`) оба бота конфликтовали:
- Ошибка: `long-polling already_started`
- Непредсказуемая обработка команд
- Потеря сообщений

## Используйте вместо этого

**Полноценный интегрированный Telegram модуль**:
- Путь: `backend/src/modules/telegram/`
- Документация: `/TELEGRAM_MODULE_README.md`
- Настройка: Web-панель `/telegram/settings`

### Возможности интегрированного модуля

✅ **База данных** - хранение пользователей, настроек, логов
✅ **Верификация** - связь Telegram с аккаунтами VendHub
✅ **Настройки уведомлений** - персонализация для каждого пользователя
✅ **QR-код линковка** - удобное подключение
✅ **API интеграция** - полный доступ к системе
✅ **Аудит** - логирование всех сообщений

### Как активировать

1. Создать бота через [@BotFather](https://t.me/BotFather)
2. Открыть админ-панель `/telegram/settings`
3. Вставить токен
4. Активировать бот
